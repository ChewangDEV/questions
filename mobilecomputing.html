Write the application of mobile computing?

Mobile computing is used in various fields due to its portability and convenience. It facilitates communication through voice calls, video conferencing, emails, and instant messaging. It provides easy access to the internet, allowing browsing and cloud services like Google Drive. Mobile banking and payment apps enable secure financial transactions. GPS-based services assist with navigation, ride-sharing, and food delivery. Mobile computing is also utilized in healthcare for telemedicine and fitness tracking, as well as in e-commerce for online shopping. Additionally, it supports remote data collection in fields like sales, delivery, and surveys.

What are the limitation of mobile computing?

Mobile computing, while offering great convenience, has several limitations. One of the major challenges is battery life, as constant use of mobile apps and services drains the battery quickly. Limited processing power compared to desktop computers can affect the performance of resource-intensive tasks. Connectivity issues like poor network coverage or unreliable Wi-Fi can hinder smooth data transmission and access. Security concerns also arise due to the risks of data breaches, hacking, and unauthorized access, as mobile devices are more vulnerable to attacks. Lastly, small screen size and input limitations (like touchscreens) can make complex tasks harder to perform compared to larger devices like laptops or desktops.

Explain prolog statements which is used in WML

In WML (Wireless Markup Language), the prolog is a required section at the beginning of a WML document. It provides important information to the WAP browser about the document type and character encoding.
Here are the key Prolog statements used in WML:

  1. XML declaration:
    <?xml version="1.0"?>
    • Declares that the document is an XML document.
    • Specifies the version of XML being used (typically 1.0).
  2. DOCTYPE Declaration:
    <!DOCTYPE wml PUBLIC "-//WAPFORUM//DTD WML 1.1//EN" "http://www.wapforum.org/DTD/wml_1.1.xml">
    • Declares the Document Type Definition (DTD) used for WML.
    • Tells the browser to use the WML 1.1 standard.
    • Ensures the document is valid according to WML rules.
  3. Root Element:
    <wml>
    <!-- WML content goes here -->
    <wml>
    • <wml> is the root element of a WML document.
    • All WML content (such as cards and tasks) must be placed within this tag.
